# Title
OpenGL 3D Cafe

# Description
Cafe scene made using OpenGL graphics library.

# Features

`▪ Custom 3D models of objects`

`▪ Custom bitmap textures for objects`
  
`▪ Keyboard controls to move camera`

`▪ Dynamic lighting for illuminating scene`
